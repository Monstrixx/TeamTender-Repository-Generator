# TeamTender Repository Generator (TRG)

TRG adalah alat untuk menghasilkan struktur repository TeamTender secara otomatis.

## Tujuan

Menghilangkan proses membuat folder dan file secara manual.

## Versi

v0.1 Alpha

## Fitur yang direncanakan

- Generate struktur repository
- Generate README
- Generate LICENSE
- Generate CONTRIBUTING
- Generate CHANGELOG
- Generate GitHub Actions
- Template enterprise
